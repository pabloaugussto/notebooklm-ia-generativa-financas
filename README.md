# Caderno Temático com NotebookLM – IA Generativa

## Descrição do Projeto
Este projeto foi desenvolvido como parte do desafio "Acelere sua Aprendizagem com IA", utilizando o NotebookLM como ferramenta de apoio ao estudo.

O desafio propôs a criação de um caderno temático a partir de três a cinco fontes abertas sobre um tema financeiro introdutório, com o objetivo de aplicar IA generativa como ferramenta de aprendizagem ativa.

## Metodologia
- Curadoria de fontes abertas (textos e PDFs)
- Definição de objetivos de estudo
- Elaboração de perguntas estratégicas
- Testes de variações de prompts
- Análise das respostas e referências fornecidas pela IA

## Resultados
- Resumos estruturados
- Glossário de conceitos
- Conjunto de prompts reutilizáveis para revisão
- Organização do conhecimento com apoio de IA

## Observação
Todo o conteúdo foi produzido no NotebookLM. Este repositório tem como finalidade documentar o processo e os resultados do estudo.
